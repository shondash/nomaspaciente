# NOMs Vigentes Aplicables

Las Normas Oficiales Mexicanas (NOMs) son de cumplimiento **obligatorio**. Violar una NOM es una infraccion administrativa sancionable (LGS Arts. 416-427). No son sugerencias.

## Que puedes hacer con una NOM

Cuando un hospital viola una NOM, tienes base para:
1. **Queja interna** — "La NOM-027 obliga a este hospital a [requisito especifico]"
2. **Queja ante Secretaria de Salud** — La Secretaria es quien aplica sanciones por violacion de NOMs
3. **Evidencia en CONAMED** — La NOM es el estandar contra el cual se mide la calidad de atencion
4. **Fundamento para amparo** — La NOM establece derechos exigibles

## Verificacion de vigencia

Vigencia confirmada mediante Aviso CNIC 4/1SE-2024 (agosto 2024) publicado en el DOF, que confirma el estatus vigente de las NOMs del sector salud.
