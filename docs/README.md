## Nivel de confianza

Cada documento marca el nivel de confianza de sus citas:

- **ALTA** — Verificado contra PDF oficial (DOF, diputados.gob.mx, IMSS, ISSSTE)
- **MEDIA** — Confirmado en fuentes secundarias; requiere verificacion contra texto oficial antes de uso en quejas formales
- **BAJA** — Referencia informativa unicamente; no usar como base legal
